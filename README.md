 🛒 Customer Segmentation using K-Means Clustering

 📌 Project Overview

This project focuses on **Customer Segmentation** using the **K-Means Clustering algorithm**.

It groups customers of a retail store based on their purchasing behavior such as **Age, Income, and Spending Score**.

An interactive **Streamlit web application** is built to visualize customer clusters and analyze patterns easily.
